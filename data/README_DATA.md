# data

Raw input files for the project.

## Files
- `movies.csv` — movieId, title, genres
- `ratings.csv` — userId, movieId, rating, timestamp
- `tags.csv` — userId, movieId, tag, timestamp
- `NLP-mergedDB.csv` — text data for NLP
- `monthly_time_series.csv` — monthly series for forecasting

## Format
- Encoding: UTF-8
- Delimiter: `,`

## Rules
- Do not write outputs here.
- Models and reports go to `models/` and `out/`.
